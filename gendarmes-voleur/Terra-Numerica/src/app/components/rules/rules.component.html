Règles

Gendarmes & Voleur

But du jeu

Dans ce jeu, deux camps s’affrontent : les gendarmes doivent attraper le voleur le plus rapidement possible. Le voleur doit fuir le plus longtemps possible.

Les gendarmes jouent en premier. Chaque pion peut se déplacer vers un sommet adjacent ou rester sur sa position.

Conditions de victoire

  • Les gendarmes gagnent si le voleur est attrapé.
  • Le voleur gagne s’il survit suffisamment longtemps sans se faire attraper.
