Comment jouer ?

1. Clique sur un point bleu pour commencer ton dessin.

2. Clique sur les points d'à côté pour tracer un trait vert.

3. Le but est de repasser sur tous les traits en pointillés.

4. Attention : tu ne peux pas passer deux fois au même endroit.

5. Essaie de finir le niveau avec 0 "levé de crayon". Si tu te trompes, tu peux toujours annuler avec le bouton ↩️.
